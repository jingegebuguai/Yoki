# @yoks/core

A lightweight and flexible state management library that serves as the foundation for @yoks.

## Features

- 🎯 Simple and intuitive API
- 🔄 Reactive state updates
- 🎨 Framework agnostic
- 📦 Tiny size
- 💪 TypeScript support
- 🔍 Proxy-based state tracking

## Installation

```bash
npm install @yoks/core
```
